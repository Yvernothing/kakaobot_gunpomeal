### 이래저래 말하고 싶은 것들

> **자세한 사항:**
> 
> - 직접 구현한 기능
> 1. 급식 한줄평
> - 기존 코드에서 발전시킨 기능
> 1. 파싱 후 html태그 제거(replace -> 정규표현식)

> 1. 장고 프레임워크
